# 🖼️ 应用图标使用指南

**创建时间**: 2026-03-13 14:01  
**尺寸**: 100x100 像素

---

## 📱 QQ 互联应用图标

### 方案 1：手动保存（推荐）

1. **右键保存你发的图片**
   - 在图片上右键 → "图片另存为"
   - 保存到：`daily-blog/public/qq-app-icon.png`

2. **调整尺寸为 100x100**
   
   **方法 A：在线工具（最简单）**
   - 打开：https://www.iloveimg.com/zh-cn/调整图像大小
   - 上传图片
   - 设置宽度 100px，高度 100px
   - 下载保存

   **方法 B：Windows 画图**
   - 右键图片 → 打开方式 → 画图
   - 点击"重新调整大小"
   - 选择"像素"，输入 100 x 100
   - 保存

   **方法 C：FFmpeg（如果你会用）**
   ```bash
   ffmpeg -i input.jpg -vf "scale=100:100" -q:v 2 qq-app-icon.png
   ```

3. **上传到 QQ 互联**
   - 回到 QQ 互联申请页面
   - 点击"上传网站图标"
   - 选择处理好的 `qq-app-icon.png`

## ✅ 图标要求

| 要求 | 说明 |
|------|------|
| 尺寸 | 100x100 像素 |
| 格式 | PNG 或 JPG |
| 大小 | < 100KB |
| 内容 | 清晰可识别，不要太复杂 |

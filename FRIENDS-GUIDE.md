## 📋 功能概览

每日博客好友系统提供完整的社交功能，包括：

- ✅ 好友列表管理（在线/离线）
- ✅ 推荐好友系统
- ✅ 好友请求处理
- ✅ 搜索好友功能
- ✅ 添加新好友
- ✅ 删除好友
- ✅ 好友聊天功能

### 2. 推荐好友

**功能：**
- 显示系统推荐的好友
- 显示共同好友数量
- 一键关注/添加

**操作：**
| 操作 | 方式 | 结果 |
|------|------|------|
| 关注 | 点击「关注」按钮 | 添加到好友列表 |
| 取消 | 点击「已关注」按钮 | 取消关注状态 |

**示例：**
```
推荐好友
┌──────────────────────────────┐
│ 👤 推荐好友 1                 │
│    共同好友：3                │
│    [关注] ← 点击添加          │
├──────────────────────────────┤
│ 👤 推荐好友 2                 │
│    共同好友：5                │
│    [已关注] ← 点击取消        │
└──────────────────────────────┘
```

### 4. 好友请求

**查看请求：**
1. 切换到「请求」标签
2. 红色角标显示未处理数量
3. 每个请求显示请求时间

**处理请求：**
| 操作 | 按钮 | 结果 |
|------|------|------|
| 接受 | 🟢 接受 | 添加到好友列表 |
| 拒绝 | ⚪ 拒绝 | 删除请求 |

**示例：**
```
请求 ②
┌──────────────────────────────┐
│ 👤 请求用户 1                 │
│    10 分钟前                   │
│    [接受] [拒绝]              │
├──────────────────────────────┤
│ 👤 请求用户 2                 │
│    1 小时前                    │
│    [接受] [拒绝]              │
└──────────────────────────────┘
```

### 5. 搜索好友

**功能：**
- 实时搜索过滤

3. 清空搜索框显示全部


### 6. 添加新好友

**方式：**
1. 点击底部「添加好友」按钮
2. 或点击推荐好友的「关注」按钮

**添加流程：**
```
1. 点击「添加好友」
   ↓
2. 输入好友名称
   ↓
3. 按 Enter 或点击「添加」
   ↓
4. ✅ 添加成功提示
   ↓
5. 自动添加到好友列表
```

**弹窗界面：**
```

├──────────────────────────────┤
│                              │

│  │ 请输入好友名称        │   │
│  └──────────────────────┘   │
│                              │
│  💡 提示：输入好友名称后，   │
│     系统将自动生成头像       │
│                              │
├──────────────────────────────┤
│      [ 添加好友 ]            │
└──────────────────────────────┘
```

### 7. 删除好友

**操作方式：**
1. 找到要删除的好友
2. 点击右侧的 🗑️ 删除按钮
3. 确认删除对话框
4. ✅ 删除成功

**确认对话框：**
```
┌──────────────────────────────┐
│  ⚠️ 确定要删除这个好友吗？    │
├──────────────────────────────┤
│      [取消] [确定]            │
└──────────────────────────────┘
```

## 📊 数据结构

### Friend 接口
```typescript
interface Friend {
  id: number;              // 唯一 ID
  name: string;            // 昵称
  avatar: string;          // 头像 URL
  status: 'online' | 'offline';  // 在线状态
  lastSeen: string;        // 最后在线时间
  isFollowing: boolean;    // 是否关注
}
```

### RecommendedFriend 接口
```typescript
interface RecommendedFriend {
  id: number;              // 唯一 ID
  name: string;            // 昵称
  avatar: string;          // 头像 URL
  commonFriends: number;   // 共同好友数
  isFollowing: boolean;    // 是否关注
}
```

### FriendRequest 接口
```typescript
interface FriendRequest {
  id: number;              // 唯一 ID
  name: string;            // 昵称
  avatar: string;          // 头像 URL
  requestTime: string;     // 请求时间
}
```

## 🐛 常见问题

### Q1: 好友列表为空？
**A:** 检查 `friends` 状态初始化，或添加新好友。

### Q2: 搜索不生效？
**A:** 确保输入了正确的关键词，搜索是实时过滤。

### Q3: 删除好友无响应？
**A:** 确认对话框需要点击「确定」才会删除。

### Q4: 如何添加真实用户？
**A:** 当前是演示数据，需要后端 API 支持。

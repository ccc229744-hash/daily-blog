## ⚠️ 潜在问题

### 1. 代码复用性

**问题：**
```tsx
// 多个组件有相似的按钮样式
<button className="px-3 py-1 bg-blue-500 text-white rounded-full text-xs">
```

**建议：**
```tsx
// 提取为可复用组件
<Button variant="primary" size="sm">
```

**优先级：** 🟡 中

---

### 2. 硬编码数据

**问题：**
```tsx
const [posts] = useState<Post[]>([
  {
    id: '2026-03-12-hello-world',
    likes: 1234,  // 硬编码
  }
])
```

**建议：**
```tsx
// 从 API 获取数据
const { data: posts } = useSWR('/api/posts')
```

**优先级：** 🟢 高

---

### 3. 缺少错误处理

**问题：**
```tsx
// 分享功能
try {
  await navigator.share(...)
} catch (err) {
  // 简单 alert
  alert('链接已复制')
}
```

**建议：**
```tsx
// 完善的错误处理
catch (err) {
  if (err.name === 'AbortError') return
  if (err.name === 'NotAllowedError') {
    showToast('分享被拒绝')
  }
  copyToClipboard()
}
```

**优先级：** 🟡 中

# OpenClaw Mac 版本的部署

## 核心观点

OpenClaw Mac 版本部署需要安装 Node.js，配置 npm 镜像源（可选），然后按照步骤安装并接入大模型能力和 channel（如飞书）。

## 主要内容

### 前置说明

**注意**：该教程请配合教学视频一起看
**视频地址**：https://www.bilibili.com/video/BV1DqfnBREgP/

### 1. 必备：安装 Node.JS

**下载地址**：https://nodejs.org/en

选择适合的 Mac 版本下载安装。

### 2. 打开终端

安装完成后，打开终端（Terminal）。

### 2.5（可选）配置 npm 镜像源 or 配置 proxy

**配置镜像源命令**：
```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com
```

### 3. 开始安装

按照官方文档执行安装命令。

### 4. 进入新手引导功能

安装完成后，进入新手引导流程。

### 5. 大模型能力接入

配置大模型 API，让 OpenClaw 具备 AI 能力。

### 6. channel 接入

配置通信渠道，如飞书、钉钉等。

### 飞书接入详细步骤（默认支持）

按照引导完成飞书机器人配置。
